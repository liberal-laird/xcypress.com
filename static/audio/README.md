# 冥想播放器应用 - 音频资源

## 音频资源说明

本应用设计支持两种类型的音频资源：

### 本地音频 (推荐用于生产环境)
- 存储在 `assets/audio/` 目录下
- 支持常见格式：MP3, WAV, AAC, OGG
- 适用于预装的冥想音频，无需网络即可播放

### 网络音频
- 通过URL播放，节省应用包体积
- 适用于动态更新的音频内容

## 预设的冥想音频分类
- 清晨冥想
- 深度放松
- 自然之音
- 专注力训练
- 睡眠引导
- 正念呼吸

## 音频时长选项
- 5分钟 - 快速放松
- 10分钟 - 中等冥想
- 15分钟 - 深度冥想
- 20分钟 - 完整冥想

## 音频资源获取建议
由于版权原因，本项目不含实际音频文件。开发者应自行获取授权的冥想音频或录制原创内容。
